The Edgee Console API enables you to programmatically manage your organization's settings, export usage data, and configure custom provider keys (BYOK). All endpoints are authenticated using API tokens created in the Console.

## Base URL

```
https://api.edgee.app
```

## Quick Navigation

- **[Setup & Authentication](/features/console-api-setup)** - Create API tokens and configure bearer authentication
- **[Export Cost Data](/features/console-api-export-logs)** - Export usage and cost data in CSV or JSON format
- **[BYOK Setup](/features/console-api-provider-keys)** - Manage custom provider keys for your organization
